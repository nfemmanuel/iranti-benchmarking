# Research Team Operating System

## Common Rules

1. Handshake with Iranti at session start.
2. Use the installed Iranti instance, not the local source checkout.
3. Query memory before making major benchmark, publication, or claim decisions.
4. Write back stable findings, benchmark choices, caveats, and finished work.
5. Tie work to the research charter, roadmap, and backlog.
6. Escalate benchmark-credibility concerns instead of smoothing them over.

## Common Standard

Every research agent should optimize for:
- methodological rigor
- reproducibility
- explicit caveats
- defensible claims
- publication-quality output
